🛍️ Crie Catálogos e Páginas de Vendas do Zero — Sem Saber Programar

Neste curso você vai aprender, passo a passo, a montar uma página de catálogo online (index) e um painel de controle (admin) para lojas, lanchonetes e materiais de construção — usando apenas ferramentas gratuitas e Inteligência Artificial. Tudo explicado de forma simples, do básico ao avançado, para computador e celular.

🆓 100% Gratuito 📱 Funciona no Celular 🤖 Usa Inteligência Artificial 🎯 Do Zero ao Avançado

1 Fundamentos: o que é Catálogo e Página de Vendas

Entenda os conceitos antes de colocar a mão na massa

Antes de criar qualquer coisa, você precisa entender duas peças fundamentais que vamos construir juntos:

🖼️
index.html
Página do Catálogo
⚙️
admin.html
Painel de Controle

📖 O que é o "index" (Catálogo)?

É a página que o cliente vê. Nela aparecem as fotos dos produtos, nomes, preços e descrições, organizados de forma bonita, com um botão para fazer o pedido (geralmente pelo WhatsApp).

⚙️ O que é o "admin" (Painel de Controle)?

É a página que só o dono do negócio acessa. Nela é possível adicionar produtos novos, mudar preços, trocar fotos, remover itens que acabaram — tudo sem precisar mexer em código.

💡 Pense assim: o index é a vitrine da loja, e o admin é o estoque/escritório nos fundos, onde você organiza tudo que vai aparecer na vitrine.

🏪 Para quais negócios isso serve?

  • Lojas (roupas, calçados, acessórios, presentes)
  • Lanchonetes e restaurantes (cardápio digital)
  • Materiais de construção (catálogo de produtos e preços)
  • Qualquer negócio que venda produtos com foto, nome e preço
📝 Tarefa do módulo: Pense em qual negócio você vai usar como exemplo durante o curso (o seu, ou de um cliente) e anote: nome do negócio, 5 produtos principais e os preços. Vamos usar isso nos próximos módulos.

3 Primeiros Passos: Entendendo o HTML do Jeito Simples

Sem termos complicados — só o essencial para você criar sua página

HTML é a "receita" que o navegador lê para montar a página. Pense nele como blocos de Lego: cada bloco (chamado de tag) tem uma função.

🏷️
<h1> = Título
📄
<p> = Texto/Parágrafo
🖼️
<img> = Imagem
🔘
<button> = Botão
📦
<div> = Caixa/Bloco

🔧 Como criar seu primeiro arquivo

1
Abra o Bloco de Notas (Windows) ou um app de edição de texto simples no celular.
2
Cole o código de exemplo abaixo.
3
Salve o arquivo com o nome index.html (é importante terminar com .html).
4
Dê dois cliques no arquivo (ou abra com o navegador) para ver o resultado.
📄 exemplo-simples.html
<!DOCTYPE html>
<html>
<head>
  <title>Minha Primeira Página</title>
</head>
<body>
  <h1>Olá, esta é minha loja!</h1>
  <p>Em breve meu catálogo completo estará aqui.</p>
</body>
</html>
💡 Dica: Não precisa decorar nada disso! Nos próximos módulos você vai copiar códigos prontos e apenas trocar os textos, preços e imagens.
📱 Pelo celular: use o app gratuito "Code Editor" (Android) ou "Textastic"/"Koder" (iPhone) para criar e salvar arquivos .html diretamente no celular.

4 Criando o Catálogo — index.html

A vitrine que seus clientes vão ver

Agora vamos montar a página real do catálogo. Esse código já vem pronto, organizado e comentado — você só precisa copiar, colar e personalizar.

🏪
LOJA EXEMPLO
🟦 [Foto] Produto 1 — R$ 29,90 — Comprar
🟩 [Foto] Produto 2 — R$ 49,90 — Comprar
🟧 [Foto] Produto 3 — R$ 19,90 — Comprar
Exemplo visual de como o catálogo vai ficar organizado em "cards" (cartões)

📋 O código completo do index.html

Esse catálogo busca os produtos automaticamente do painel admin (módulo 5). Ou seja: tudo que você cadastrar no admin aparece aqui sozinho, sem precisar editar o código de novo.

📄 index.html
<!DOCTYPE html>
<html lang="pt-BR">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Catálogo - Nome da Loja</title>
<style>
  body { font-family: Arial, sans-serif; background:#f4f4f4; margin:0; }
  header { background:#1e3a8a; color:#fff; padding:20px; text-align:center; }
  header h1 { margin:0; font-size:24px; }
  #busca { width:90%; max-width:400px; padding:10px; margin:16px auto; display:block; border-radius:8px; border:1px solid #ccc; }
  #produtos { display:grid; grid-template-columns:repeat(auto-fit,minmax(220px,1fr)); gap:16px; padding:20px; max-width:1100px; margin:0 auto; }
  .card { background:#fff; border-radius:12px; overflow:hidden; box-shadow:0 2px 8px rgba(0,0,0,0.1); }
  .card img { width:100%; height:180px; object-fit:cover; }
  .card-info { padding:14px; }
  .card-info h3 { margin:0 0 6px; font-size:16px; }
  .card-info p { color:#666; font-size:13px; margin-bottom:10px; }
  .preco { color:#1e3a8a; font-weight:bold; font-size:18px; margin-bottom:10px; }
  .btn-comprar { display:block; text-align:center; background:#25D366; color:#fff; padding:10px; border-radius:8px; text-decoration:none; font-weight:bold; }
</style>
</head>
<body>

  <header>
    <h1>🏪 Nome da Loja</h1>
    <p>Catálogo Online</p>
  </header>

  <input type="text" id="busca" placeholder="🔍 Buscar produto...">
  <div id="produtos"></div>

<script>
  const numeroWhatsApp = "5511999999999"; // troque pelo seu número com DDD

  function carregarProdutos() {
    const produtos = JSON.parse(localStorage.getItem('produtos')) || [];
    const container = document.getElementById('produtos');
    const termo = document.getElementById('busca').value.toLowerCase();
    container.innerHTML = '';

    produtos
      .filter(p => p.nome.toLowerCase().includes(termo))
      .forEach(p => {
        const link = `https://wa.me/${numeroWhatsApp}?text=Olá! Tenho interesse no produto: ${p.nome} (R$ ${p.preco})`;
        container.innerHTML += `
          <div class="card">
            <img src="${p.imagem}" alt="${p.nome}">
            <div class="card-info">
              <h3>${p.nome}</h3>
              <p>${p.descricao}</p>
              <div class="preco">R$ ${p.preco}</div>
              <a class="btn-comprar" href="${link}" target="_blank">💬 Pedir no WhatsApp</a>
            </div>
          </div>`;
      });
  }

  document.getElementById('busca').addEventListener('input', carregarProdutos);
  carregarProdutos();
</script>

</body>
</html>
⚠️ Importante: Troque 5511999999999 pelo número de WhatsApp real, com código do país (55) + DDD + número, sem espaços ou símbolos.
💡 Esse código usa o localStorage, que é uma "gaveta" de memória do navegador. Os produtos cadastrados no admin ficam guardados ali e o catálogo lê essas informações automaticamente.

5 Criando o Painel Admin — admin.html

Onde você controla tudo o que aparece no catálogo

Esse é o "painel de controle". Nele você adiciona produto, define nome, preço, descrição e imagem (a foto é convertida automaticamente para que funcione sem precisar de servidor).

⚙️
PAINEL ADMIN
➕ Nome do produto: ____________
💲 Preço: ____________
📝 Descrição: ____________
📷 [Escolher Imagem]  [Salvar Produto]
Formulário simples para cadastrar cada item do catálogo
📄 admin.html
<!DOCTYPE html>
<html lang="pt-BR">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Painel Admin - Catálogo</title>
<style>
  body { font-family: Arial, sans-serif; background:#f4f4f4; margin:0; padding:20px; }
  h1 { color:#1e3a8a; }
  form { background:#fff; padding:20px; border-radius:12px; max-width:500px; margin-bottom:30px; box-shadow:0 2px 8px rgba(0,0,0,0.1); }
  label { display:block; margin-top:12px; font-size:14px; font-weight:bold; color:#333; }
  input, textarea { width:100%; padding:10px; margin-top:4px; border:1px solid #ccc; border-radius:8px; font-size:14px; }
  button { margin-top:16px; background:#1e3a8a; color:#fff; padding:12px; border:none; border-radius:8px; width:100%; font-size:15px; cursor:pointer; }
  .lista-produto { background:#fff; padding:12px; border-radius:10px; margin-bottom:10px; display:flex; gap:12px; align-items:center; max-width:600px; box-shadow:0 1px 4px rgba(0,0,0,0.08); }
  .lista-produto img { width:60px; height:60px; object-fit:cover; border-radius:8px; }
  .lista-produto .info { flex:1; }
  .lista-produto button { width:auto; background:#ef4444; padding:8px 14px; margin:0; font-size:13px; }
</style>
</head>
<body>

  <h1>⚙️ Painel de Controle do Catálogo</h1>

  <form id="form-produto">
    <label>Nome do produto</label>
    <input type="text" id="nome" required>

    <label>Preço (somente números, ex: 29,90)</label>
    <input type="text" id="preco" required>

    <label>Descrição curta</label>
    <textarea id="descricao" rows="2"></textarea>

    <label>Imagem do produto</label>
    <input type="file" id="imagem" accept="image/*" required>

    <button type="submit">➕ Salvar Produto</button>
  </form>

  <h2>📦 Produtos Cadastrados</h2>
  <div id="lista"></div>

<script>
  function getProdutos() { return JSON.parse(localStorage.getItem('produtos')) || []; }
  function salvarProdutos(lista) { localStorage.setItem('produtos', JSON.stringify(lista)); }

  function renderizarLista() {
    const produtos = getProdutos();
    const lista = document.getElementById('lista');
    lista.innerHTML = '';
    produtos.forEach((p, index) => {
      lista.innerHTML += `
        <div class="lista-produto">
          <img src="${p.imagem}">
          <div class="info"><strong>${p.nome}</strong><br>R$ ${p.preco}</div>
          <button onclick="excluirProduto(${index})">🗑️ Excluir</button>
        </div>`;
    });
  }

  function excluirProduto(index) {
    const produtos = getProdutos();
    produtos.splice(index, 1);
    salvarProdutos(produtos);
    renderizarLista();
  }

  document.getElementById('form-produto').addEventListener('submit', function(e) {
    e.preventDefault();
    const nome = document.getElementById('nome').value;
    const preco = document.getElementById('preco').value;
    const descricao = document.getElementById('descricao').value;
    const arquivo = document.getElementById('imagem').files[0];

    const leitor = new FileReader();
    leitor.onload = function() {
      const produtos = getProdutos();
      produtos.push({ nome, preco, descricao, imagem: leitor.result });
      salvarProdutos(produtos);
      renderizarLista();
      e.target.reset();
    };
    leitor.readAsDataURL(arquivo);
  });

  renderizarLista();
</script>

</body>
</html>
💡 Como funciona a mágica: ao salvar um produto no admin, ele entra na mesma "gaveta" (localStorage) que o index.html lê. Por isso é essencial que os dois arquivos (index.html e admin.html) fiquem na mesma pasta e no mesmo site publicado.
⚠️ Limitação importante: o localStorage guarda os dados só naquele navegador/computador. Para um catálogo "de verdade" que sincroniza em qualquer aparelho, é preciso evoluir para um banco de dados online (vamos falar disso no Módulo 11 - Avançado).

6 Usando IA Gratuita para Criar Textos e Descrições

Nunca mais fique sem saber o que escrever

Use o ChatGPT ou Google Gemini (gratuitos) para gerar nomes chamativos, descrições e até sugestões de preço. Veja modelos de prompts (instruções) prontos para copiar:

🍔 Prompt para Lanchonete

Crie uma descrição curta e apetitosa (até 20 palavras) para um
hambúrguer artesanal com carne 180g, queijo cheddar, bacon e
molho especial. Use linguagem que dê vontade de comprar.

👕 Prompt para Loja de Roupas

Escreva uma descrição de venda, curta e atrativa, para uma
camiseta básica de algodão, cor branca, modelo unissex.
Foque no conforto e versatilidade.

🧱 Prompt para Material de Construção

Crie uma descrição técnica e simples para um saco de cimento
50kg, explicando a aplicação principal (alvenaria e reboco)
em até 25 palavras.
💡 Dica de ouro: peça para a IA gerar "5 variações diferentes" de uma vez. Assim você escolhe a que mais combina com o seu negócio.

📋 Prompt para gerar uma lista inteira de produtos

Tenho uma [tipo de negócio]. Crie uma tabela com 10 produtos
comuns desse tipo de negócio, incluindo: nome do produto,
descrição curta e faixa de preço sugerida no Brasil.
📱 Pelo celular: abra o app do ChatGPT ou Gemini, cole o prompt, copie a resposta e cole direto no campo "Descrição" do seu Painel Admin.

7 Usando IA Gratuita para Imagens de Produtos

Fotos profissionais sem precisar de fotógrafo

1
Tire a foto do produto com boa luz natural (perto de uma janela, de dia).
2
Acesse remove.bg pelo navegador, envie a foto e baixe a versão sem fundo (gratuito).
3
Abra o Canva, crie um design com fundo bonito (cor lisa ou textura) e cole o produto sem fundo por cima.
4
Use a ferramenta de IA do Canva ("Magic Studio") para gerar fundos ou melhorar a iluminação automaticamente.
5
Baixe a imagem final e use o TinyPNG para deixá-la leve (carregamento rápido na página).

🪄 Não tem fotos? Crie imagens com IA

Caso ainda não tenha o produto fotografado (ex: está só planejando o catálogo), você pode usar geradores de imagem por IA gratuitos como o Bing Image Creator ou o Canva IA, descrevendo o produto em texto.

Exemplo de prompt: "Foto realista de um hambúrguer artesanal
suculento, fundo branco, estilo fotografia de cardápio,
iluminação profissional"
⚠️ Imagens 100% geradas por IA podem não representar o produto real — use preferencialmente fotos reais sempre que possível, principalmente em alimentos.

8 Modelos Prontos por Tipo de Negócio

Ajustes de cor, ícone e textos para cada segmento

🍔 Lanchonete / Restaurante

  • Cor sugerida: vermelho/laranja (#dc2626) — transmite apetite
  • Use categorias: Lanches, Bebidas, Sobremesas
  • Inclua tempo estimado de entrega na descrição

👗 Loja de Roupas / Acessórios

  • Cor sugerida: preto/dourado ou rosa, conforme o público
  • Inclua tamanhos disponíveis na descrição (P, M, G)
  • Fotos em fundo branco neutro, sempre no mesmo padrão

🧱 Material de Construção

  • Cor sugerida: laranja/cinza (#ea580c) — remete a obra
  • Inclua unidade de medida (saco, m², unidade, metro)
  • Organize por categoria: Cimento, Tintas, Elétrica, Hidráulica
💡 Para trocar a cor principal, basta alterar o valor de background:#1e3a8a no <style> do index.html pelo código de cor do seu negócio.

🗂️ Adicionando categorias no catálogo (avançado simples)

Você pode adicionar um campo "categoria" no formulário do admin e um filtro no index, repetindo a mesma lógica do campo "busca" já criado no Módulo 4.

9 Publicando sua Página Gratuitamente

Colocando o catálogo no ar com um link real

💻 Pelo Computador — usando Netlify

1
Acesse netlify.com e crie uma conta gratuita (pode usar o e-mail ou Google).
2
Coloque os arquivos index.html e admin.html dentro de uma mesma pasta no computador.
3
No painel do Netlify, procure a opção "Deploy manually" e arraste a pasta para a área indicada.
4
Em poucos segundos, o Netlify gera um link (ex: seunegocio.netlify.app).
5
Para acessar o admin, basta visitar seunegocio.netlify.app/admin.html.
💡 É possível personalizar o nome do site nas configurações do Netlify, gratuitamente, em "Site settings > Change site name".

📱 Pelo Celular

1
Salve os dois arquivos (.html) em uma pasta no app de Arquivos do celular.
2
Acesse netlify.com pelo navegador do celular e entre na sua conta.
3
Use a opção de upload de pasta/arquivos (em alguns celulares pode ser necessário compactar a pasta em .zip antes).
4
Aguarde a publicação e copie o link gerado.

🌐 Alternativa: GitHub Pages

Outra opção 100% gratuita é o GitHub Pages: crie uma conta no GitHub, suba os arquivos em um repositório e ative o "Pages" nas configurações. O processo é parecido e também gera um link público.

⚠️ Guarde bem o link do admin.html — ele não tem senha por padrão neste modelo simples. Para proteger, use o mesmo sistema de senha desta própria página que você está vendo agora (login com senha simples em JavaScript).

10 Botão de Pedido Direto no WhatsApp

Transformando visitas em vendas de verdade

O botão "Pedir no WhatsApp" já está incluso no código do Módulo 4. Ele funciona criando um link especial que abre o WhatsApp já com uma mensagem pronta, citando o produto escolhido.

🔗 Estrutura do link mágico do WhatsApp
https://wa.me/NUMERO?text=MENSAGEM
  • NUMERO: código do país (55) + DDD + número, sem espaços (ex: 5511999999999)
  • MENSAGEM: o texto que já aparece pronto no WhatsApp do cliente
💡 Você pode personalizar a mensagem para incluir automaticamente o nome e o preço do produto, exatamente como já está programado no código do Módulo 4.

📲 Configurando o WhatsApp Business (recomendado)

1
Baixe o app WhatsApp Business (gratuito).
2
Configure mensagens automáticas de boas-vindas para quando o cliente clicar no botão.
3
Cadastre o catálogo de produtos também dentro do próprio WhatsApp Business como reforço.

11 Nível Avançado

Evoluindo seu catálogo para um sistema profissional

☁️ Banco de dados online (sincronizar entre aparelhos)

O localStorage funciona só no navegador onde foi cadastrado. Para que o admin atualize o catálogo em qualquer dispositivo, o próximo passo é usar um banco de dados gratuito como o Google Firebase (plano gratuito) ou Supabase.

  • Crie uma conta gratuita no Firebase ou Supabase
  • Troque as funções localStorage.setItem e getItem por chamadas à API do banco (a própria IA, como ChatGPT, pode te ajudar a adaptar o código colando-o e pedindo: "adapte este código para usar Firebase no lugar de localStorage")

🛒 Carrinho de compras com vários produtos

Permite que o cliente escolha vários produtos antes de enviar um único pedido pelo WhatsApp, somando os valores automaticamente.

🔎 SEO básico (aparecer no Google)

  • Adicione no <head> a tag <meta name="description" content="..."> com uma frase descrevendo seu negócio
  • Use um título (<title>) claro, com o nome do negócio e a cidade
  • Cadastre seu site gratuitamente no Google Meu Negócio

📐 Categorias e filtros

Adicione um campo "categoria" no admin e crie botões de filtro no topo do catálogo (ex: Todos | Lanches | Bebidas), filtrando o array de produtos por essa categoria, da mesma forma que já fizemos com a busca por texto.

🔐 Protegendo o admin com senha

Use exatamente o mesmo sistema de senha desta página do curso (tela de login em JavaScript) na frente do seu admin.html, para impedir que outras pessoas mexam no catálogo.

💡 Sempre que travar em algo técnico nesse nível avançado, copie seu código atual, cole no ChatGPT/Gemini e descreva exatamente o que você quer adicionar. A IA ajuda a adaptar o código sem você precisar saber programar profundamente.

12 Como Vender Esse Serviço Para Outras Pessoas

Transformando esse conhecimento em renda

Depois de dominar a criação de catálogos, você pode oferecer esse serviço para comerciantes locais que ainda não têm presença online.

💰 Como precificar

  • Criação do catálogo: cobre um valor fixo pela montagem inicial (design, cadastro de produtos, publicação)
  • Manutenção mensal: ofereça um plano para atualizar produtos e preços todo mês
  • Pacote completo: inclua tratamento de imagens com IA e textos prontos como diferencial

📣 Onde encontrar clientes

  • Pequenos comércios do seu bairro que ainda usam apenas papel ou WhatsApp simples
  • Grupos de empreendedores no Facebook e WhatsApp
  • Indicação de clientes satisfeitos

🎯 Argumento de venda simples

"Eu monto um catálogo online para o seu negócio, onde o
cliente vê os produtos, preços e já manda o pedido direto
no seu WhatsApp — sem mensalidade de aplicativo caro,
e você mesmo consegue atualizar preços quando quiser."
💡 Use esta própria página como exemplo de portfólio: mostre o catálogo funcionando ao vivo para o cliente em potencial, no celular dele.
